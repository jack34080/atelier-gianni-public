# 📝 Bloc-Notes – Manuel d’utilisation

## Présentation
Un éditeur de texte riche avec dossiers, correcteur orthographique et export.

## Prérequis
- Atelier GIANNI P. installé

## Utilisation
1. Ouvrez le module.
2. Le panneau de gauche affiche vos **dossiers** et **notes**.
3. Cliquez sur **📁+** pour créer un dossier, **📄+** pour une note.
4. Rédigez dans la zone d’édition à droite.

## Barre d’outils
| Bouton | Fonction |
|--------|----------|
| **G** | Gras |
| *I* | Italique |
| <u>S</u> | Souligné |
| Normal / Titre 1/2/3 | Styles de titre |
| • | Liste à puces |
| ⫷ / ⬜ / ⫸ | Alignement |
| ✅ | Correcteur orthographique |
| 📥 TXT / MD | Export |

## Recherche
Cliquez sur **🔍** pour ouvrir la barre de recherche et filtrer les notes.

## Panneau latéral
Cliquez sur **☰** pour masquer/afficher le panneau des dossiers.

## Problèmes courants
| Problème | Solution |
|----------|----------|
| Correcteur ne souligne rien | Vérifiez que le bouton ✅ est actif |
