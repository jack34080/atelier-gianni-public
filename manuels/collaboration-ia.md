# 🔄 Collaboration IA – Manuel d’utilisation

## Présentation
Deux IA égales collaborent pour produire un contenu enrichi sur plusieurs itérations.

## Prérequis
- Atelier GIANNI P. installé
- Une clé API DeepSeek valide
- Une licence active pour le module Collaboration IA

## Utilisation
1. Ouvrez le module.
2. Saisissez un sujet et cliquez sur **🚀 Lancer**.
3. **IA1** produit une première version.
4. **IA2** l’enrichit avec de nouveaux éléments.
5. Le résultat final combine les deux contributions.

## Réglages
- **Modèle** : choisissez le modèle pour chaque IA.
- **Itérations** : nombre d’allers-retours (1 à 5).
- **Clés API** : testez et validez chaque clé.

## Problèmes courants
| Problème | Solution |
|----------|----------|
| Une IA ne répond pas | Vérifiez sa clé API avec le bouton **Tester** |
