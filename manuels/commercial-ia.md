# 💼 Commercial IA – Manuel d’utilisation

## Présentation
Un module complet avec six fonctions : prospects, forums, stock, GitHub, mises à jour et téléchargements.

## Prérequis
- Atelier GIANNI P. installé
- Une clé API DeepSeek valide
- Un token GitHub (pour la partie GitHub)
- Une licence active pour le module Commercial IA

## Les onglets
| Onglet | Fonction |
|--------|----------|
| 🔍 Prospects | L’IA recherche des prospects pour vos produits |
| 📢 Forums | L’IA rédige des messages promotionnels |
| 📊 Stock | Gérez vos produits et surveillez les alertes de stock |
| 📦 GitHub | Déployez automatiquement sur votre dépôt GitHub |
| 🔄 MàJ | Vérifiez les mises à jour disponibles |
| 📥 Téléchargements | Compteur de téléchargements |

## Gestion du stock
1. Remplissez les champs en bas : Produit, Prix, Quantité.
2. Choisissez Entrée ou Sortie, ajoutez une raison.
3. Cliquez sur **+ Stock**.
4. Les produits sous le seuil (5 unités) déclenchent une alerte rouge.

## Déploiement GitHub
1. Configurez votre token GitHub dans **Paramètres**.
2. Cliquez sur **📦 Déployer sur GitHub (réel)**.
3. Le module pousse automatiquement une mise à jour.

## Problèmes courants
| Problème | Solution |
|----------|----------|
| Token GitHub manquant | Configurez-le dans **Paramètres** |
| Déploiement échoué | Vérifiez que le dépôt existe et que le token a les droits |
