# 🤝 Duo IA – Manuel d’utilisation

## Présentation
Le Duo IA fait collaborer deux intelligences artificielles : un **Architecte** qui propose des solutions, et un **Critique** qui les évalue et les améliore.

## Prérequis
- Atelier GIANNI P. installé
- Une clé API DeepSeek valide
- Une licence active pour le module Duo IA

## Prise en main
1. Ouvrez le module via le Chat (**📦 Modules** → **Duo IA**) ou via l’Orchestrateur.
2. Vérifiez que les voyants sont verts (connexion à l’API OK).
3. Dans la zone de saisie en bas, tapez votre sujet et cliquez sur **🚀 Lancer**.

## Fonctionnement
- L’**Architecte** (à gauche) rédige une première réponse détaillée.
- Le **Critique** (à droite) analyse cette réponse et l’enrichit.
- Le processus se répète sur plusieurs itérations (réglable avec le champ **🔄 Itérations**).

## Les sélecteurs
- Vous pouvez choisir le **modèle d’IA** pour chaque rôle (DeepSeek, OpenAI, etc.).
- Les champs de clé API permettent d’utiliser une clé différente pour chaque IA.

## Clic droit
- Sur un message : **📋 Copier** ou **💾 Télécharger** le texte.

## Problèmes courants
| Problème | Solution |
|----------|----------|
| Voyant rouge | Vérifiez la clé API dans le champ correspondant |
| Pas de réponse | Cliquez sur **Tester** pour vérifier la connexion |
