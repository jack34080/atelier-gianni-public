# 🧾 Facturation – Manuel d’utilisation

## Présentation
Créez des devis et des factures professionnels, avec un assistant IA pour vous aider.

## Prérequis
- Atelier GIANNI P. installé
- Une clé API DeepSeek valide (pour l’assistant IA)
- Une licence active pour le module Facturation

## Créer un document
1. Cliquez sur **📄 Devis** ou **🧾 Facture** dans le panneau de gauche.
2. Remplissez les champs : N°, Date, Client, Type.
3. Ajoutez des lignes avec le bouton **+ Ajouter une ligne**.
4. Remplissez la description, la quantité et le prix unitaire.
5. Les totaux (HT, TVA, TTC) se calculent automatiquement.

## Assistant IA
Dans le panneau du bas, vous pouvez dialoguer avec l’IA pour :
- Générer un numéro de facture
- Calculer des remises
- Vérifier la cohérence du document
- Créer automatiquement des lignes

Exemple : *« Crée un devis pour une prestation de formation de 3 jours à 500 € par jour »*

## Export
Utilisez l’impression du navigateur (Ctrl+P) pour exporter en PDF.

## Problèmes courants
| Problème | Solution |
|----------|----------|
| L’IA ne répond pas | Vérifiez la licence dans Paramètres |
