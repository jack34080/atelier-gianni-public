# 🤖 IA Indépendante – Manuel d’utilisation

## Présentation
Une IA autonome, avec un historique de conversation et des commandes spéciales.

## Prérequis
- Atelier GIANNI P. installé
- Une clé API DeepSeek valide
- Une licence active pour le module IA Indépendante

## Utilisation
1. Ouvrez le module.
2. Choisissez un modèle dans le sélecteur en haut.
3. Tapez votre question et appuyez sur **▶ Envoyer**.
4. L’IA répond dans la zone principale.

## Commandes spéciales
| Commande | Action |
|----------|--------|
| `/memo texte` | Sauvegarde un mémo |
| `/memoire` | Affiche les 5 derniers mémos |
| `/help` | Affiche l’aide |
| 🗑️ | Efface l’historique |

## Historique
L’IA garde en mémoire les 10 derniers échanges pour contextualiser ses réponses.

## Problèmes courants
| Problème | Solution |
|----------|----------|
| Clé API non configurée | Enregistrez une clé dans le module **Paramètres** |
