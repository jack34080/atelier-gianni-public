# 🔔 Notifications – Manuel d’utilisation

## Présentation
Consultez l’état de l’Atelier, les tests fonctionnels et les actions automatiques.

## Prérequis
- Atelier GIANNI P. installé

## Utilisation
1. Ouvrez le module.
2. Cliquez sur **🔄 Actualiser** pour charger les dernières informations.
3. Consultez les sections :
   - État de l’orchestrateur
   - Tests fonctionnels
   - Actions automatiques

## Problèmes courants
| Problème | Solution |
|----------|----------|
| Impossible de charger | Vérifiez que l’Atelier est bien lancé |
