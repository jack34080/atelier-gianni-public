# 🔎 Veille – Manuel d’utilisation

## Présentation
Prenez des notes de veille stratégique, avec analyse par IA.

## Prérequis
- Atelier GIANNI P. installé
- Une clé API DeepSeek valide
- Une licence active pour le module Veille

## Utilisation
1. Ouvrez le module.
2. Cliquez sur **+ Nouvelle note**.
3. Donnez un titre et rédigez votre contenu.
4. Cliquez sur **💾 Enregistrer**.

## Analyse IA
Dans le champ en bas, demandez à l’IA d’analyser votre note :
- *« Résume cette note »*
- *« Quelles tendances se dégagent ? »*
- *« Propose des actions à mener »*

## Problèmes courants
| Problème | Solution |
|----------|----------|
| Note non sauvegardée | Cliquez sur 💾 après chaque modification |
