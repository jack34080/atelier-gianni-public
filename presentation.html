⚙️ Comment ça marche ?

1

Téléchargez l'application (Pack Solo gratuit).

2

Lancez l'Atelier. Le Chat s'ouvre.

3

Saisissez votre clé API DeepSeek (ou OpenAI, Anthropic, Google).

4

Activez votre licence dans le module Paramètres.

5

Utilisez les modules achetés. L'IA est intégrée partout.

L'Orchestrateur (🧠) supervise tous les modules en temps réel : tests, alertes, workflows.
